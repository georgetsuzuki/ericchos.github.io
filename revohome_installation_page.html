REVO HOME

Installation Instructions

Kit Contents

The REVO HOME Kit includes the following

  • 9K-Module
  • Wi-Fi Receiver
  • RFC Remote
  • Wiring Harnesses & Mounting Hardware

Plug-In REVO HOME Wi-Fi Receiver

Plug the REVO Home Wi-Fi Receiver into a wall socket in a location with a strong Wi-Fi signal and a strong signal to the Fireplace. You can test the signal integrity by standing where you plan on installing the REVO HOME Wi-Fi receiver and testing the fireplace with the RFC remote.

Available as a Kit

Kit includes:

  • 9K-Module
  • Wi-Fi Receiver
  • RFC Remote
  • Wiring Harnesses & Mounting Hardware

Want Flexibility?

The REVO HOME system also include Dexen's RFC remote so you can have a second remote option. Multiple iOS devices can be used with REVO HOME.
